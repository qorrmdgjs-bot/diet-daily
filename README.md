# Diet Daily � (귀여운 아기공룡과 함께)

다이어트 중인 사용자를 위한 체중 관리 웹 애플리케이션

## 기능

- **온보딩**: 처음 앱 실행 시 체중 입력 페이지로 이동
- **체중 입력**: 아침/저녁 몸무게를 날짜별로 입력 (소수점 1자리)
- **과거 데이터 입력**: 날짜 선택으로 과거 체중 데이터 추가 가능
- **체중 추이 차트**: recharts를 사용한 선 그래프, 7일/30일/전체 필터, 목표 체중 점선, 7일 이동 평균선
- **예측 기능**: 최근 7일 데이터를 기반으로 선형 회귀 계산, 목표 도달 예측
- **귀여운 아기공룡 마스코트 (옆모습 일러스트)**: 상황별 랜덤 멘트 출력, 말풍선 UI
- **대시보드**: 오늘 체중, 변화량, 주간 감량, 목표까지 남은 kg, 스트릭, BMI, 변동폭
- **설정 페이지**: 목표 체중과 키 설정
- **추가 기능**: BMI 자동 계산, CSV 내보내기, 다크모드 지원

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- recharts (차트)
- date-fns (날짜 처리)
- next-themes (다크모드)
- localStorage (데이터 저장)

## 설치 및 실행

```bash
npm install
npm run dev
```

## 사용법

1. 처음 실행 시 체중 입력 페이지가 나타납니다.
2. 목표 체중과 키를 설정하고 오늘 체중을 입력하세요.
3. "시작하기" 버튼을 누르면 대시보드 페이지로 이동합니다.
4. 매일 체중을 입력하고 차트를 확인하세요.
5. 설정 페이지에서 목표를 변경할 수 있습니다.
6. 귀여운 아기공룡의 응원을 받으세요!
